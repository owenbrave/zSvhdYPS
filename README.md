# 项目简介

基于pyqt和深度学习的吸烟智能检测系统，主要用于监控公共场所吸烟行为。系统包含实时视频流处理、图像识别、吸烟行为检测等功能模块。技术特点在于采用深度学习算法，实现对吸烟行为的精准识别，具有良好的实时性和准确性。通过pyqt框架构建的用户界面，便于操作人员进行系统管理和监控。此系统不涉及售卖、交易或服务性话术，专注于技术实现与应用。


## 核心功能
- 实时视频流处理
- 图像识别
- 吸烟行为检测
- 用户界面管理

## 适用场景
- 公共场所吸烟监控
- 系统管理与监控

## 技术特点
- 采用深度学习算法实现精准识别
- 具有良好的实时性和准确性
- 基于pyqt框架构建的用户界面
- 专注于技术实现与应用
